# Vehicle Rental System (Java + Oracle JDBC)

## Description
A console-based Vehicle Rental System developed using Java and basic JDBC.
The project demonstrates how Java applications interact with an Oracle database
to perform CRUD operations.

## Technologies
- Java
- JDBC (Oracle)
- SQL
- OOP Concepts

## Features
- View available vehicles
- Rent a vehicle
- Return a rented vehicle
- Database-based storage using JDBC

## Project Structure
- model : Entity classes
- dao   : JDBC database operations
- util  : Database connection
- Main  : Menu-driven application

## JDBC Details
- Used DriverManager for Oracle JDBC connection
- Used PreparedStatement for executing SQL queries
- Followed DAO pattern
